This is a rudimentary tool developed to help making backgrounds a bit easier.

Initial Sprite Index is the sprite of the first block placed. The sprite increments by one for every block placed horizontally.
Horizontal Length is how many blocks are placed horizontally.
Vertical Length is how many blocks are placed vertically, starting from the top.
Layer, is the layer used to place the blocks on. Defaults to 0.
X Offset is the horizontal offset of the blocks placed. 1 is one tile, 2 is two tiles, etc..
Y Offset is the vertical offset, same principle. 1 is one tile, etc..

Press P to activate preview mode.
Left-click selects, middle-click creates a new warp, WASD keys to move, delete to delete.
For tiles that have a warp, the extra data of that tile should match the ID of the warp point.
For example, a warp pipe with extra data 32 will warp to the warp point in that level with the ID 32.
The extra data of the warp point defines the animation state.
0 is default. 1, 2, 3, and 4 are warp pipe up, down, left and right respectively.
You can also use 5 and 6 to use up and down respectively without the player being offset by half a tile.
